CORPUS FILES DIRECTORY
======================

This directory contains corpus text files that will be automatically loaded by the application.

HOW TO ADD NEW CORPUS FILES:
-----------------------------

1. Create a new .txt file in this directory
2. Name it descriptively (e.g., "Book Title - Author Name.txt")
3. The filename (without .txt) will appear in the dropdown menu
4. Save your text content in UTF-8 encoding
5. Refresh the web application to see your new corpus

NAMING EXAMPLES:
----------------
- "Pride and Prejudice - Jane Austen.txt"
- "Things Fall Apart - Chinua Achebe.txt"
- "The Joy Luck Club - Amy Tan.txt"
